🚀 Fresh Termux Setup for Node.js + PM2

This guide helps you set up a new Termux install with Node.js, npm, PM2, and your GitHub Bash script.

✅ 1. Update Termux & Install Basics

pkg update && pkg upgrade -y
pkg install curl git unzip -y

🔧 2. Install Node.js via NVM

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc

💡 If ~/.bashrc doesn’t exist, try source ~/.profile or run:

source ~/.nvm/nvm.sh

🟢 3. Install Node.js & npm Using NVM

nvm install --lts
nvm use --lts

⚙️ 4. Install PM2

npm install -g pm2

📦 5. Download & Extract Your GitHub Repo

curl -L https://github.com/qpixio/az-printer-service/archive/refs/heads/main.zip -o repo.zip
unzip repo.zip
cd az-printer-service-main

🚀 6. Run Your Bash Script

bash setup.sh

📁 (Optional) Enable Storage Access

termux-setup-storage

✅ Summary

  • Node + npm: via nvm
  • PM2: global install
  • Repo: downloaded + extracted ZIP
  • Script: launched with bash setup.sh
